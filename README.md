# Student Management System

## 📌 Description
A Java-based Student Management System developed using NetBeans.

## ✨ Features
- Add Student
- Update Student
- Delete Student
- Search Student

## 🛠️ Technologies Used
- Java
- NetBeans IDE

## ▶️ How to Run
1. Open the project in NetBeans.
2. Build the project.
3. Run the application.
